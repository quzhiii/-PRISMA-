## Export Boundary

Final result exports are conflict-gated in V2.5:

- `included_studies.csv`
- `excluded_studies.csv`
- PRISMA SVG exports
- `screening_report.md`
- `quality_appraisal.csv`
- `evidence_table.csv`
- `grade_summary.csv`
- `prisma_counts.json`
- `audit_summary.md`

When unresolved conflicts remain, these exports are blocked and an `export_conflict_blocked` audit event is recorded.

Evidence exports remain available so reviewers can inspect and resolve conflicts:

- `project_manifest.json`
- `events.jsonl`
- `screening_decisions.csv`
- `exclusion_reasons.csv`
- `ai_usage_registry.json`
- `ai_suggestions.jsonl`
- `PRISMA_TRAICE_REPORT.md`
- `dual_review_conflicts.csv`
- `dual_review_agreement.json`

When unresolved conflicts remain, these evidence exports show a warning and record `export_conflict_warning`.

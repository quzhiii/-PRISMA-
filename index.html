📖 常用数据库文献导出指南

🔬 PubMed 导出步骤
  1. 在PubMed搜索框输入检索式，点击"Search"
  2. 点击"Send to"按钮
  3. 选择"Citation manager"
  4. 点击"Create file"，下载.nbib文件（即RIS格式）
  5. 将下载的文件直接拖到上方上传区域

💡 推荐格式：RIS (.nbib) - 包含完整摘要和关键词

📚 知网(CNKI) 导出步骤
  1. 在知网检索结果页面，勾选要导出的文献
  2. 点击"导出/参考文献"
  3. 选择"Endnote"格式
  4. 点击"导出"，保存.enw文件
  5. 将文件上传到本工具

💡 推荐格式：Endnote (.enw) - 本工具已完美支持中文

🌐 Web of Science 导出步骤
  1. 在检索结果页面，勾选要导出的文献（最多500条）
  2. 点击"导出"按钮
  3. 文件格式选择"其他文件格式"
  4. 记录内容选择"完整记录和引用的参考文献"
  5. 文件格式选择"纯文本"或"RIS"
  6. 点击"导出"，下载文件

💡 推荐格式：RIS - 数据最完整

📖 Cochrane Library 导出步骤
  1. 搜索并打开检索结果
  2. 勾选要导出的文献
  3. 点击"Export selected"
  4. 选择"RIS"格式
  5. 下载并上传文件
🔧 使用文献管理软件（推荐）

使用 Zotero（免费推荐）：

  1. 将所有数据库的文献导入Zotero
  2. 选中要导出的文献
  3. 右键 → "导出条目"
  4. 格式选择"RIS"
  5. 保存并上传到本工具

⭐ 优势：Zotero可以自动去重、管理文献，大大提高效率！

字段映射

系统会自动将不同格式的字段映射到标准字段：

  • title: RIS(T1/TI) → BibTeX(title) → CSV(title/Title)
  • abstract: RIS(AB) → BibTeX(abstract) → CSV(abstract)
  • year: RIS(PY) → BibTeX(year) → CSV(year)
  • authors: RIS(AU) → BibTeX(author) → CSV(authors)
  • journal: RIS(T2/JO) → BibTeX(journal) → CSV(journal)
  • doi: RIS(DO) → BibTeX(doi) → CSV(doi)
